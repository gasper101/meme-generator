# meme-generator

Za zagon aplikacije samo poženite docker container. Nato pa lahko stran obiščete na localhost:5000.
To lahko storite tako da se postavite v koren mape, ter zaženete ukaz: 'docker build -t meme-generator'.
Nato pa 'docker run -p 5000:5000 meme-generator' za zagon containerja.

Na strani nato najprej naložite sliko, ter vpišete zgornje in spodnje besedilo, ter kliknete generate.
Spodaj se bo nato prikazala nova slika.
